<h1>Kodregler</h1>

<ul>
  <li>Alla klassnamn, metoder och variabler ska skrivas på engelska</li>
  <li>Tydliga och beskrivande namn för klasser, metoder och variabler</li>
  <li>Beskriv relevant kod genom kommentarer</li>
  <li>Kommentarer ska vara skrivna på svenska</li>
  <li>Standard-casing för Java ska alltid användas</li>
  <li>Commits ska tydligt beskrivas</li>
  <li>Kod ska diskuteras tillsammans innan den mergas till main-branch</li>
  <li>All versionshantering ska ske genom Git och Github</li>
</ul>
